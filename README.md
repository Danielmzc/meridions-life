# Meridions Life

Site institucional da Meridions Life, desenvolvido com Next.js, React e Tailwind CSS.

## Rodando localmente

```bash
npx pnpm@latest install
npx pnpm@latest dev
```

Abra `http://localhost:3000`.

## Deploy na Vercel

Framework preset: `Next.js`

Build command:

```bash
npx pnpm@latest build
```

Install command:

```bash
npx pnpm@latest install --frozen-lockfile
```
